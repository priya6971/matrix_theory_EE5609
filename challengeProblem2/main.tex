\section{Problem}
Prove that the eigenvalues of $\vec{A^TA}$ are positive. 
\section{Proof}
Let, $\vec{A}$ is an arbitrary $m\times n$ matrix. Now consider the matrix $\vec{A^TA}$,
\begin{align}
\intertext{for any n dimensional vector $\vec{x}$,}
\vec{x^T}(\vec{A^TA})\vec{x} &= \vec{x^TA^T}\vec{Ax}\\
\implies\vec{x^T}(\vec{A^TA})\vec{x} &= \vec{({Ax})^T}(\vec{Ax})\\
\implies\vec{x^T}(\vec{A^TA})\vec{x} &= \norm{\vec{Ax}}^2 \geq0\label{eq1}
\intertext{From \eqref{eq1}, if $\vec{x} \not=0$, $\vec{A^TA}$ is positive definite, i.e}
\norm{\vec{Ax}}^2 > 0
\intertext{Again, $\vec{A^TA}$ is positive semi-definite, if $\vec{x} = 0$,}
\norm{\vec{Ax}}^2 = 0
\end{align}
Hence, $\vec{A^TA}$ is positive semi-definite if the columns of $\vec{A}$ are linearly dependent and $\vec{A^TA}$ is positive definite if columns of $\vec{A}$ are linearly dependent.\\
Again,
\begin{align}
\vec{({A^TA})^T} = \vec{(A^T)(A^T)^T} = \vec{A^TA}\label{eqSym}
\end{align}
Hence, $\vec{A^TA}$ is symmetric. As every eigen value of a Hermitian matrix is real and every symmetric matrix is Hermitian then $\vec{A^TA}$ has real eigen values.\\
Let $\lambda$ be a eigenvalue of $\vec{M}=\vec{A^TA}$ and let $\vec{x}$ be a corresponding real eigen-vector hence,
\begin{align}
\vec{Mx} = \lambda \vec{x}\label{eq2}
\intertext{Multiplying $\vec{x^T}$ in \eqref{eq2},}
\vec{x^TMx} &= \lambda \vec{x^T}\vec{x}\\
\implies\vec{x^TMx} &= \lambda \norm{\vec{x}}^2\\
\implies\vec{x^T(A^TA)x} &= \lambda \norm{\vec{x}}^2\label{eqFinal} 
\end{align}
When $\vec{A^TA}$ is positive definite (i.e columns of $\vec{A}$ are linearly independent) then $\vec{x}$ is a nonzero vector as it is an eigen-vector.
Since $\norm{\vec{x}}^2$ is positive, hence all eigen-values must be positive.
Hence proved.
